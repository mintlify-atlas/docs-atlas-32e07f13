<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold dark:text-white text-gray-900 mb-4">Quick start</h2>
  <p className="dark:text-gray-400 text-gray-600 mb-8">Get up and running with DZDK CLI in minutes</p>

  <Steps>
    <Step title="Install the CLI">
      Clone the repository and install dependencies:
      ```bash
      git clone https://github.com/Dzaleka-Connect/dzdk-cli.git
      cd dzdk-cli
      python3 -m venv venv
      source venv/bin/activate
      pip install -e .
      ```
    </Step>

    <Step title="Configure the API endpoint">
      Set your API URL and timeout preferences:
      ```bash
      dzdk config --url "https://services.dzaleka.com/api" --timeout 30
      ```
      <Info>Configuration is stored in `~/.config/dzdk/config.yaml`</Info>
    </Step>

    <Step title="Check API health">
      Verify your connection to the API:
      ```bash
      dzdk health
      ```
      This command checks all endpoints and displays response times and status codes.
    </Step>

    <Step title="List available services">
      Browse services in the Dzaleka camp:
      ```bash
      dzdk services list --page 1
      ```
      You'll see a formatted table with service details, categories, and contact information.
    </Step>
  </Steps>
</div>
